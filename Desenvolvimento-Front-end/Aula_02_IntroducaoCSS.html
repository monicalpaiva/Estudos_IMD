Aula 02 - Introdução ao CSS

    1. Entender o CSS e sua aplicação

      2. O CSS (Cascade Style Sheets) é uma linguagem que auxilia a estilização de páginas HTML. Enquanto o HTML se preocupa com a organização e estruturação do conteúdo, o CSS se preocupa com a aparência desse conteúdo.

      O CSS pode ser incluído no documento HTML de várias formas:

      • A mais simples é escrever o código CSS no cabeçalho do documento HTML dentro das tags style e /style
      • É possível importar um arquivo CSS de um arquivo externo. Para importar um arquivo CSS em uma página HTML, utilize a tag link
      • Outra forma é indicar um código CSS diretamente na tag HTML em que se deseja aplicar o estilo. O CSS foi definido através da propriedade style (EX.: style="background-color: #FF0000")
    2. Entender como as cores são representadas

      3. Atualmente existem 140 cores que podem ser definidas pelos seus respectivos nomes. Para a lista completa das cores disponíveis, consulte css_colors

      Representações:

      • background-color: "green";
      • background-color: rgb(0, 255, 0);
      • background-color: #00FF00;
