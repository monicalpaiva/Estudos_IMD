Aula 12 - HTML Responsivo

O que é responsividade

A responsividade é a capacidade do site se adaptar a diferentes dispositivos.

Viewport

O primeiro conceito que você precisa entender para trabalhar com responsividade é o conceito de Viewport. O Viewport é a área disponível para o site no dispositivo. Diferentes dispositivos têm diferentes tamanhos de viewport. Para criar sites responsivos é importante seguir algumas dicas:

  • Não use elementos grandes com largura fixa;
  • Não permita que o conteúdo dependa de determinada largura de viewport para renderizar bem;
  • Aplique diferentes estilos para diferentes tamanhos de tela.

Imagens responsivas

Um desses controles veio através da tag <meta name="viewport" content="width=device-width, initial-scale=1.0">. Ao adicionar essa estrutura no cabeçalho do documento HTML, estamos informando ao navegador a escala do viewport, o que gera melhor aparência para diferentes dispositivos.

Importante: Por padrão, ao adicionar uma imagem ao HTML, ela é apresentada em seu tamanho real, sem qualquer redimensionamento. Para deixá-la mais responsiva, você pode definir o tamanho da imagem de forma relativa. No entanto, a definição relativa através dos atributos "width" e "heigth", pode gerar perda de qualidade da imagem - quando esta fica é apresentada em um tamanho maior que o real. Para resolver isso, o CSS criou um atributo chamado "max-width", que permite definir o tamanho da imagem relativo, mas respeitando o tamanho máximo real da imagem. Outra forma de controlar melhor a imagem, a depender da resolução, é utilizar a tag <picture>, que permite definir diferentes imagens para diferentes resoluções de tela.


<picture>
<source srcset="img/imagem-small.jpg" media="(max-width: 600px)" />
<source srcset="img/imagem-med.jpg" media="(max-width: 1200px)" />
<source srcset="img/imagem-large.jpg" />
<img src="img/imagem-med.jpg" alt="">
</picture>

Textos responsivos

É importante conhecer uma nova unidade que permite definir o tamanho da fonte de forma relativa ao viewport. Essa configuração pode ser feita através da utilização da unidade "vw". Essa unidade permite definir um valor de tamanho relativo ao tamanho do viewport, deixando a aparência do texto melhor adaptada ao dispositivo.

A medida "vw" é igual a 1/100 da largura da viewport. Então, por exemplo, se a largura do navegador é 900px, 1vw equivale a 9px. Analogamente, há também a unidade "vh", que está relacionada à altura do viewport. Dessa forma, se a altura da viewport é 750px, 1vh equivale a 7.5px.

Media Queries

Essa funcionalidade foi introduzida na versão 3 do CSS e facilitou a aplicação de estilos com base na resolução de tela disponível.


@media only screen and (max-width: 600px) {
body {
background-color: rebeccapurple;
}

@media only screen and (min-width: 1200px) {
body {
background-color: red;
}

Diferentes condições podem ser utilizadas (orientação da tela, resolução etc). Além disso, é permitido o uso de múltiplas condições através de operadores lógicos. Para ver mais exemplos das possibilidades, consulte este link.
