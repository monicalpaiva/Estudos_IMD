Aula 07 - Formulários

Criando formulário

Os formulários são dispositivos que permitem receber entrada do usuário no HTML. Para criar um formulário, deve-se utilizar a tag <form>.

Entrada de texto

Para adicionar um campo de entrada de texto no formulário, deve-se utilizar a tag <input> com atributo "type" igual a "text".

Rótulos

Os rótulos permitem informar aos usuários o que se espera que seja preenchido em determinado campo do formulário. Para adicionar um rótulo a um campo de entrada, utiliza-se a tag <label>, indicando no atributo "for" o identificador do campo de entrada associado.

Botão de submissão

Os botões de submissão permitem aos usuários enviar os dados do formulário. Para adicionar um botão de submissão, utilize a tag <input> com o atributo "type" igual a "submit".

Ação de submissão

A ação de submissão deve ser informado como o atributo "action" da tag <form>. Nesse campo deve ser informado o endereço do servidor que contém um programa responsável por capturar esses dados e realizar alguma ação.

Código Exemplo:

<form action="https://autoria-form.herokuapp.com/" method="GET">
<label for="nome">Nome:</label>
<input type="text" id="nome" name="nome">
<label for="sobrenome">Sobrenome:</label>
<input type="text" id="sobrenome" name="sobrenome"/>
<input type="submit" value="Enviar dados">
</form>

Saída do Exemplo:






O método GET utiliza o próprio endereço (URL) para transferir os dados ao servidor. Além do método GET, existe o método POST, que envia os dados no corpo da requisição HTTP e que por isso não aparece na URL ao usuário.

Select, Textarea e Datalist

Campo de seleção

O campo de seleção (select) permite mostrar ao usuários uma lista de opções para que ele possa escolher uma ou mais opções. Para criar um campo select, utilize a tag <select>. Por padrão o campo de seleção permite selecionar apenas uma opção. Caso queira mudar esse padrão para permitir selecionar múltiplos, basta adicionar o atributo ""multiple"".



Área de texto

O campo área de texto (textarea), permite que o usuário insira um texto longo de vários parágrafos. Para adicionar um campo de área de texto, utilize a tag <textarea>

<textarea name="redacao" rows="10" cols="30"></textarea>

Datalist

O datalist é uma ferramenta que permite criar uma lista de opções de entrada para um campo de texto. Para criar um datalist, utilize a tag <datalist> com cada opção descrita pela tag <option>.

Aplicando estilo aos formulários

A estilização de formulários segue o mesmo principio dos demais componentes HTML.







Dimensão

Você pode controlar a largura/altura dos campos de formulário através dos atributos CSS "width" e "height". As medidas podem ser absolutas ou relativas.

O atributo "box-sizing" permite configurar como a propriedade de tamanho será aplicada ao componente. Por padrão, ao definir uma largura/altura, o CSS aplica esse tamanho ao componente sem considerar espaçamentos e borda. Para que o CSS considere a borda e os espaçamentos, deve-se utilizar o valor "border-box" para essa propriedade. Para mais opções, veja este link.

Alteração de fonte dos labels

Para alterar as fontes do label, pode-se utilizar o seletor de tag e aplicar determinado estilo de fonte, assim como em qualquer outro componente.

Seleção de tag e atributo

O CSS permite selecionar todas as tags que possuem valor em uma propriedade. Essa funcionalidade é especialmente importante nos formulários, já que diferentes tipos de entradas de dados são indicados pelo atributo "type" do campo .

input[type=text] {
padding: 12px;
margin: 8px 0px;
border: 1px solid lightgray;
border-radius: 4px;
}

input[type=submit] {
width: 100%;
background-color: #4CAF50;
color: white;
padding: 10px;
border-radius: 4px;
cursor: pointer;
}

A propriedade "cursor" permite controlar o cursor do mouse que será mostrado ao usuário. No exemplo acima, o cursor padrão foi alterado para o valor "pointer", indicado quando se deseja mostrar que um componente é clicável. Para mais opções de cursor, veja este link.
