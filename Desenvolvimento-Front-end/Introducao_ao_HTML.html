Aula 01 - Introdução ao HTML

    1. Conhecer o HTML e sua história

      2. O HTML é uma linguagem de marcação utilizada na construção de páginas na web. Ou seja, o HTML não é uma linguagem de programação em que o programador tem liberdade de programar comportamentos a partir da entrada de dados. Em vez disso, as linguagens de marcação descrevem informações de apresentação para que possam ser apresentadas aos usuários. O HTML é hoje utilizado por todo e qualquer site na internet para a criação e estruturação do seu conteúdo.

      O HTML foi criado por um pesquisador do CERN, Tim Berners-Lee em 1991, com o objetivo de facilitar o compartilhamento de pesquisas entre ele e seu grupo de colegas.

      O primeiro navegador foi criado pelo próprio Tim e ficou conhecido como World Wide Web. Em 1993 nasce o primeiro navegador gráfico, o Mosaic.

      Gerra dos navegadores

    2. Conhecer os principais editores de texto

      3. O Sublime é um editor de texto leve e fácil de usar. Para o uso é necessário adquirir uma licença.

      O Atom é um editor de texto e código-fonte gratuito compatível com macOS, Linux e Microsoft Windows com suporte para plug-ins escritos em Javascript, desenvolvido pelo GitHub.

      O Visual Studio Code, ou somente VSCode, é um editor de texto gratuito, de código aberto, desenvolvido pela Microsoft compatível com sistema operacional Windows, Linux e macOS.

    3. Conhecer a estrutura básica de um documento HTML!

      4. A primeira linha indica a versão do documento HTML, o conteúdo deve ser <!DOCTYPE html">.

      Em seguida, dá-se início à estruturação do HTML. Para isso, os demais elementos HTML, devem ficar entre os elementos <html> e </html>.

      Para incluir informações de cabeçalho como, por exemplo, o título e os metadados, deve-se informar esses dados entre os elementos <head> e </head>.

      Por fim, o conteúdo a ser mostrado pelo navegador ao usuário, deve ficar delimitado entre os elementos <body> e </body>.

Aula 02 - Introdução ao CSS

    1. Entender o CSS e sua aplicação

      2. O CSS (Cascade Style Sheets) é uma linguagem que auxilia a estilização de páginas HTML. Enquanto o HTML se preocupa com a organização e estruturação do conteúdo, o CSS se preocupa com a aparência desse conteúdo.

      O CSS pode ser incluído no documento HTML de várias formas:

      • A mais simples é escrever o código CSS no cabeçalho do documento HTML dentro das tags style e /style
      • É possível importar um arquivo CSS de um arquivo externo. Para importar um arquivo CSS em uma página HTML, utilize a tag link
      • Outra forma é indicar um código CSS diretamente na tag HTML em que se deseja aplicar o estilo. O CSS foi definido através da propriedade style (EX.: style="background-color: #FF0000")
    2. Entender como as cores são representadas

      3. Atualmente existem 140 cores que podem ser definidas pelos seus respectivos nomes. Para a lista completa das cores disponíveis, consulte css_colors

      Representações:

      • background-color: "green";
      • background-color: rgb(0, 255, 0);
      • background-color: #00FF00;

Aula 03 - Títulos, Parágrafos, Links e Imagens

    1. Conhecer como adicionar títulos e parágrafos em HTML

      2. Os títulos no HTML são utilizados para dividir o documento HTML em seções. O HTML permite organizar o conteúdo em até seis níveis. (h6 a h6).

      É importante utilizar os títulos de forma semanticamente correta. Ou seja, o título principal do seu site deve estar na tag h6. Caso deseje alterar a aparência como, por exemplo, por um título menor, utilize o CSS. Com isso, você manterá a consistência da informação, fazendo com que seu site seja corretamente interpretado por sistemas externos.

      Para criar um parágrafo em HTML, deve-se utilizar a tag "p".

    2. Conhecer como adicionar links e como estilizá-los

      3. Os links no HTML são utilizados para permitir acessar outros documentos através de um simples clique. Graças aos links, podemos ter uma melhor interação com os sistemas web.

      Usamos a tag "a" para criar links no HTML

      Valores do atributo “target”:

      • _blank: Abre o link em uma nova janela (ou aba)
      • _self: Abre o link na mesma janela que foi clicado (comportamento padrão)
      • _parent: Abre o link no frame "pai"
      • _top: Abre o link no topo da hierarquia de frames
      • framename: Abre o link no iframe cujo nome coincida com o desse parâmetro

      a href="www.link.com" target="_blank"

      Estados dos links:

      • link: Link que ainda não foi visitado
      • visited: Link que já foi visitado
      • hover: Ativo quando o usuário está com o mouse sobre o link
      • active: Ativo quando o usuário clica no link

      A definição "a:hover" deve vir após a definição "a:link" e "a:visited" para ser efetiva! Similarmente, a definição "a:active" deve vir depois da a:hover para ser efetiva.

    3. Conhecer como adicionar imagens e como estilizá-las

      4. Para adicionar imagens em um documento HTML, utilize a tag <img>.

      Especificando o caminho para a imagem através do atributo "src"

      <img src="img/html-css.jpg">

      Indicamos um texto alternativo para imagem, que será exibido quando a imagem não puder ser exibida, através do atributo "alt". Esse atributo permite deixar sua página mais acessível, pois pode ser lido por sistemas de leitura de telas, comumente utilizados por pessoas com alguma limitação visual.

      <img src="img/html-css.jpg" alt="Livro de HTML & CSS">

        OBS: Além da definição de tamanho absoluto, é possível definir a largura ou a altura através de valores relativos. Nesse caso, a imagem ocupará uma parte da área disponível.

      • Atributos de dimensão

        • Para definir a largura utilize o atributo "width" e para definir a altura, utilize o atributo "height".

      • Atributos de posição

        • Para posicionar a imagem à esquerda ou à direita, de forma flutuante, através do atributo "float".

      • Atributos de espaçamento

        • Para adicionar margem às imagens e adicionar espaçamento do conteúdo em relação à imagem, utiliza-se o atributo "margin". Para editar apenas algum dos lados podem ser usados também os atributos: "margin-top", "margin-right", "margin-bottom" e "margin-left".

Listas

Utilizam-se as tags <ul> para listas não ordenadas e a tag <ol> para listas ordenadas. As tags <li> para cada elemento.

<ol type="A">
<li>Café</li>
<li>Chá</li>
<li>Leite</li>
</ol>
    Atributos das listas numéricas no HTML
  1. Os itens serão numerados por algarismos arábicos (1)
  2. Os itens serão numerados pelas letras maiúsculas do alfabeto (A)
  3. Os itens serão numerados pelas letras minúsculas do alfabeto (a)
  4. Os itens serão numerados por algarismos romanos maiúsculos (I)
  5. Os itens serão numerados por algarismos romanos minúsculos (i)
Estilizando as listas

A estilização das listas segue a mesma forma das demais estilizações. É possível aplicar através do CSS um estilo para todas as listas, ou apenas para seus itens.

ul {
list-style-type: disc;
}
    Valores para a aparência dos marcadores
  • Os marcadores serão discos sólidos (aparência padrão) - disc
  • Os marcadores serão quadrados sólidos - square
  • Os marcadores serão círculos vazados - circle
  • Nenhum marcador será utilizado - none
