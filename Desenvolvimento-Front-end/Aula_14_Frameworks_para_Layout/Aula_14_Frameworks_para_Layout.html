O que é o Bootstrap?

Bootstrap

O Bootstrap é uma biblioteca criada para facilitar a criação de sites responsivos desenvolvida pelo Twitter e depois disponibilizada gratuitamente para a comunidade. O Bootstrap oferece um conjunto de CSS (e Javascripts) que coferem ao documento um aspecto profissional e responsivo.

Containers e Grid (parte 1)

Sistema de grid do Bootstrap.

O sistema de grid é uma importante ferramenta que facilita a estruturação do conteúdo do documento HTML. Seu funcionamento consiste em dividir o conteúdo em linhas e colunas. Você pode controlar o número de colunas, sua proporção. O comportamento é dinâmico e adaptável a diferentes resoluções.

Toda a configuração é realizada através da criação de "divs" com classes específicas do Bootstrap. As classes permitem definir linhas, colunas e seu comportamento em diferentes resoluções.

O Bootstrap permite criar um grid com no máximo 12 colunas. No entanto, é possível controlar a distribuição dos espaços entre as colunas através do uso de classes específicas. Por exemplo, se quisermos que uma coluna possua o tamanho referente a 4 espaços, devemos utilizar a classe "col-md-4" (você pode trocar o "md" por "sm", "lg", "xl" para diferentes pontos de quebra, ou até mesmo utilizar apenas "col-4").

Tabelas

Tabelas utilizando classes do Bootstrap

Perceba o uso das tags <thead> e <tbody> para descrever o cabeçalho e o corpo da tabela, respectivamente. Similarmente, existe também a tag <tfoot>, com o objetivo de descrever o rodapé da tabela, quando houver. Essas tags semânticas foram adicionadas no HTML5 e servem para descrever o conteúdo da tabela e facilitar, inclusive, a aplicação de estilo e acessibilidade.

Opções de cabeçalho

Você pode controlar a aparência do cabeçalho. Por padrão, o cabeçalho terá a aparência clara. Para alterar esse comportamento, você pode utilizar a classe "thead-dark" no elemento <thead>.

Linhas zebradas

Você pode utilizar a classe "table-striped" para tornar as linhas zebradas, facilitando a identificação das linhas.

Linhas com efeito "hover"

Você pode adicionar efeito de destaque, quando o usuário tiver com o mouse em cima de uma linha. Para isso, deve-se utilizar a classe "table-hover".

Tabela pequena

O estilo padrão das tabelas do Bootstrap possui um "padding" para espaçamento interno. Caso deseje ter uma tabela mais condensada, você pode utilizar a classe "table-sm".

Alguns componentes do Bootstrap

Barra de navegação

A barra de navegação permite criar menus responsivos.

Carousel

O Carousel é um componente de slideshow para fazer troca de elementos (imagens ou slides de texto), como se fosse um carrossel.

Cards

Os Cards Bootstrap proporcionam um contêiner de conteúdo flexível e extensível com múltiplas variantes e opções.

Sombras

Adicione ou remova sombras de elementos, usando os utilitários box-shadows. Para adicionar sombra a um elemento, basta adicionar a classe "shadow".
