Aula 06 - Tabelas

Para criar uma tabela, deve-se utilizar a tag <table>.

Para cada linha deve-se utilizar a tag <tr> e, para cada coluna, deve-se utilizar a tag <td>.

Para indicar que o dado é um cabeçalho, deve-se utilizar a tag <th>, em vez de <td>

Mesclando linhas e colunas

Mesclando colunas

Para mesclar duas ou mais colunas, utilize o atributo "colspan" da tag <td> ou <th>, indicando o número de colunas que devem ser mescladas.

<th colspan="2">Telefone</th>
Name Telefone
Bill Gates 55577854 55577855

Mesclando linhas

Para mesclar duas ou mais linhas, utilize o atributo "rowspan" na tag <td> ou <th>, indicando o número de linhas que devem ser mescladas.

<th rowspan="2">Telefone</th>
Name Bill Gates
Telefone 55577854
55577855

Aplicando estilo em uma Tabela

Espaçamento

É possível utilizar a propriedade "padding" para controlar o espaçamento entre a borda e o conteúdo da tabela. Para que isso fique visível, é importante que a borda não esteja invisível.

Alinhamento de texto

Para controlar o alinhamento do texto na célula da tabela, utilize a propriedade text-align para alinhamento horizontal e vertical-align para vertical.

Borda

Algo importante a analisar é se as bordas das tabelas ficarão condensadas ou separadas através da propriedade "border-collapse".

Cor de fundo

A cor de fundo da tabela é controlada pela mesma propriedade utilizada para outros elementos: "background-color".

É possível aplicar estilos diferentes para linhas pares e linhas ímpares, obtendo-se uma melhor separação entre as linhas.

/* par */
tr:nth-child(even) {
background-color: #eee;
}
/* ímpar */
tr:nth-child(odd) {
background-color: lightblue;
}

Este tipo de seletor avançado se chama seletor de pseudoclasse.

Assim como os links, pode-se definir o estilo para as tabelas a depender do estado.

tr:hover {
background-color: blue;
}

Melhor explicação de cada tag para tabelas:

<table> Define uma tabela

<th> Define uma célula de cabeçalho em uma tabela

<tr> Define uma linha em uma tabela

<td> Define uma célula em uma tabela

<caption> Define uma legenda de tabela

<colgroup> Especifica um grupo de uma ou mais colunas em uma tabela para formatação

<col> Especifica as propriedades da coluna para cada coluna dentro de um elemento

<thead> Agrupa o conteúdo do cabeçalho em uma tabela

<tbody> Agrupa o conteúdo do corpo em uma tabela

<tfoot> Agrupa o conteúdo do rodapé em uma tabela
