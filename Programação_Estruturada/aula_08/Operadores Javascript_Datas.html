Aula 08 - Operadores Javascript: Datas

Definindo datas

Em JavaScript, datas são objetos. A data é composta por ano, mês, dia, hora, minutos, segundos e milissegundos. Para criar objetos do tipo Data, temos que usar o construtor Date(). Além disso, como estamos criando um objeto, temos que usar a palavra-chave new antes do construtor.

var agora = new Date();
