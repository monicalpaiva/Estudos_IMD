Definindo Strings

JavaScript possui diversas outras funções sobre strings, por exemplo:
=> funções que retornam o tamanho de uma string;
=> para encontrar partes específicas dentro de uma string;
=> para extrair partes de uma string
=> para transformar strings

Para caracterizar uma string é necessário inseri-las dentro de aspas simples ou aspas duplas. Quando em uma string for necessário usar aspas para alguma palavra, deve-se usar as aspas simples na string e aspas duplas onde deseja colocar, ou vice-versa. JavaScript também permite o uso da barra invertida para esse uso.

"Um exemplo de string com \"aspas internas\".";

Lembrando que se usarmos apenas uma barra invertida ela não irá aparecer na string, é necessário utilizar duas.

"Temos uma \\ dentro da string.";

Para quebrar a linha em uma organização de código, é recomendado quebra-la após o atribuidor (=), ao quebrar a linha colococando uma \ não é recomendado, pois alguns navegadores podem não reconhecer. Podemos então quebra-la fazendo concatenação (+) de strings.

"Este texto está ficando demasiadamente \ grande para esta linha."

Em JavaScript, a comparação de strings é sensível a letras maiúsculas e minúsculas.

Para efetuar a comparação ignorando se estão em maiúsculas e minúsculas, podemos usar as funções toUpperCase() ou toLowerCase().

concat() tem a mesma função do +.

Exemplos:

Strings como Objetos
var string_1 = "IMD"
var string_2 = "I" + "M".concat("D"); // "IMD"
var string_3 = "I".concat("M","d"); // "IMd"

var objeto_1 = new String("IMD");
var objeto_2 = new String("IMD");

OBS: De uma forma geral, sempre trate números, strings e booleanos, como valores primitivos, não como objetos. Isso porque declarar esses tipos como objetos diminui a velocidade de execução e produz os efeitos colaterais desagradáveis.

Procurando e Extraindo Strings

Para descobrir a quantidade de caracteres/comprimento de uma string vamos usar a propriedade length

O primeiro método que podemos usar em uma string, o indexOf(), recebe uma string como parâmetro e retorna a posição da primeira ocorrência dessa string na string em que chamamos o método. O método search() também apresenta o mesmo comportamento.

OBS: É importante destacar o fato de que JavaScript conta posições a partir do zero.

o lastIndexOf(), retorna o índice da última ocorrência de um texto especificado em uma string.

Uma característica importante do retorno dos três métodos, indexOf(), search() e lastIndexOf(), é que, se a string não possuir o valor passado como parâmetro, todos eles retornam -1.

Apenas o método indexOf() pode ser chamado usando um segundo parâmetro que indica a partir de que posição a busca deve começar. Além disso, apenas o método search() pode ser chamado usando uma expressão regular como parâmetro. De maneira bastante resumida, uma expressão regular é um objeto que descreve um padrão de caracteres e que pode ser usado para fazer o que chamamos de casamento de padrão ou para "pesquisar e substituir" no texto.

Exemplos:
var texto = "ALÔ MUNDO MUNDO";
var resultado10 = texto.indexOf("MUNDO", 5);
var resultado11 = texto.indexOf("MUNDO", 4);
var resultado12 = texto.search("MUNDO");
var resultado13 = texto.lastIndexOf("MUNDO");
var patt = /[uo]/i;
var resultado14 = texto.search(patt);// O que estamos procurando são as letras "u" ou "o" e não considerando se é maiúscula ou minúscula.
var resultado15 = texto.lenght(texto);

Extração de partes da string

O método slice() recebe dois parâmetros, a posição inicial e a posição final, e retorna uma nova string resultante de extrair a parte da string que começa na posição inicial e termina antes da posição final. Se um parâmetro for negativo, a posição será contada a partir do final da string. Esse método também pode ser usado com apenas um parâmetro. Nesse caso, ele retorna uma nova string resultante de extrair a parte da string que começa na posição passada, caso o parâmetro seja negativo, a posição será contada a partir do final da string.

O método subString() é praticamente idêntico ao método slice(). A única diferença é que parâmetros negativos são interpretados como 0

O método substr() também é muito similar ao slice(), a diferença é que o segundo parâmetro especifica o comprimento da parte extraída. Além disso, assim como no método subString(), parâmetros negativos são interpretados como 0. Se você omitir o segundo parâmetro, substr(), cortará o restante da string. Por isso, da mesma maneira que o método slice() e o método subString()

O método charAt() retorna apenas um caractere naquela parêmetro passado.

O método split() usado para separar os elementos de uma string usando um caractere separador.

OBS: Em JavaScript strings são imutáveis, ou seja, elas não podem ser alteradas, apenas substituídas.

Exemplos:

Transformando Strings

O método replace() substitui a primeira ocorrência da string passada como primeiro parâmetro pela string passada como segundo parâmetro. Assim como nos métodos de extração de strings que vimos, o método replace() não altera a string à qual ele está sendo aplicado, mas retorna uma nova string como resultado. Esse método diferencia letras maiúsculas e letras minúsculas.

Exemplo:
var texto3 = "ALÔ MUNDO LINDO";
var resultado28 = texto3.replace("LINDO","BELO");

var resultado29 = texto;

// Substituição apenas da primeira ocorrência
var texto4 = "ALÔ MUNDO LINDO LINDO";
var resultado30 = texto4.replace("LINDO","BELO");

// Método é case sensitive
var texto5 = "ALÔ MUNDO LINDO";
var resultado31 = texto5.replace("lindo","belo");

// Expressões regulares para case insensitive
var texto6 = "ALÔ MUNDO LINDO";
var resultado32 = texto6.replace(/lindo/i,"belo");

// Expressões regulares para substituir todas as ocorrências
var texto7 = "ALÔ MUNDO LINDO LINDO";
var resultado33 = texto7.replace(/LINDO/g,"BELO");
var resultado34 = texto7.replace(/lindo/ig,"belo");

Observamos nos exemplos temos que o i ignora os casos de minúsculo e maiúsculos e o g faz a alteração de ambas as palavras no texto.
