Exercício

Utilizando o arquivo Atividade_02-2.html como base, insira, na área reservada para o código JavaScript, um código que escreva no campo minhaResposta os tipos de cada uma das expressões abaixo, um em cada linha. Não se preocupe se você não conhece algumas dessas expressões. Você as conhecerá ao longo da disciplina. Nesta atividade, concentre-se apenas em conseguir descobrir o seu tipo. Note também que nesta questão está sendo exigida a exibição com quebra de linhas. Para isso, você poderá inserir no texto resultante a tag html < br > de quebra de linhas.
