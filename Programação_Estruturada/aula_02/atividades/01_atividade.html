Atividade 01

Utilizando o arquivo Atividade_02-1.html como base, insira, na área reservada para o código JavaScript, um código que declare cinco variáveis e atribua os valores de 5, 4, 3, 2 e 1 a elas. Esse código também deve declarar outras duas variáveis e atribuir a elas os textos "Aprendi" e "JavaScript". O código JavaScript deverá escrever no campo de texto minhaResposta do html o texto "5, 4, 3, 2, 1... Aprendi JavaScript!!!". Note que foram inseridos espaços em branco no texto resultante. Portanto, a página html final deverá ter a seguinte exibição:

Atividade 02-1

5, 4, 3, 2, 1... Aprendi JavaScript!!!
