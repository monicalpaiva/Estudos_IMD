Interpretando Tipos de Dados

Da Esquerda para a Direita


  • Os tipos básicos são:
    1. [number] para números de qualquer tipo: inteiro ou ponto flutuante;
    2. [string] para strings, ou seja, para cadeias de um ou mais caracteres - em JavaScript não há um tipo de caractere único separado;
    3. [boolean] para os valores booleanos true e false;
    4. [null] para valores desconhecidos - este é um tipo independente que possui um único valor null; e,
    5. [undefined] para valores não atribuídos - este também é um tipo independente que possui um único val1or undefined.
  • Os principais tipos complexos são:
    1. [function] para funções;
    2. [symbol] para identificadores exclusivos; e,
    3. [object] para estruturas de dados mais complexas.

Operações com number

Precedência

Além dos números regulares, existem os chamados valores numéricos especiais, que também pertencem a esses tipos de dados, são eles: infinity e -infinity e , que representam os números fora do intervalo de números possíveis. E NaN, "Not a Number", que representa um número ilegal como, por exemplo, o resultado de dividir 100 pelo texto "programa".
