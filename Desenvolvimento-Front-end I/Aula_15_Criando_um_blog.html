Aula 15 - Criando um blog

Estrutura inicial

Criar a estrutura básica do site que conterá uma barra de navegação, um Breadcrumb e um rodapé.

Barra de navegação

Para a barra de navegação, utilizaremos as classes do Bootstrap e a tag semântica <nav> do HTML.

Breadcrumb

O Breadcrumb é um componente do Bootstrap que cria um localizador, permitindo ao usuário se localizar e navegar mais rapidamente pelas páginas do site.

Rodapé

Ao final do site, vamos adicionar um rodapé com informações adicionais do Blog, como, por exemplo, informações do desenvolvedor. Para isso, utilizaremos a tag semântica <footer>, com algumas classes do Bootstrap para aplicar o estilo.

Perceba o uso das seguintes classes nativas do Bootstrap:

  • py-3 - adiciona padding top (superior) e bottom (inferior) de tamanho 3.
  • text-white - torna o texto branco.
  • text-center - centraliza o texto.

Sidebar

Construção da estrutura do conteúdo principal com foco na barra lateral, chamada de sidebar. Para criar a estrutura do conteúdo e da barra lateral, utilizaremos o sistema de grid do Bootstrap.

Card de busca

No sidebar, adicionaremos uma área para o usuário buscar algum conteúdo no blog. Para isso, utilizaremos o componente Card do Bootstrap.

Card de categorias

Além do componente de busca, adicionaremos ao sidebar uma listagem das categorias do blog.

Card do autor

Por fim, adicionaremos ao sidebar um card com informações do autor. Para isso, será utilizado um componente card do Bootstrap e algumas classes utilitárias para estilização da foto e espaçamentos.

Conteúdo principal

Título

Para o título, utilizaremos a tag <h1>, que indica o título principal do documento HTML. Além disso, adicionaremos uma linha horizontal (<hr>) e alguns ajustes de estilo.

Postagens

Para cada uma das postagens, utilizaremos um Card que conterá uma imagem ilustrativa, o título do post, uma pequena descrição e informações da postagem.

Página interna

Topo

No início de cada postagem, vamos destacar o título e informações adicionais da postagem.

Corpo

O corpo da postagem deverá conter uma imagem ilustrativa e o conteúdo da postagem. Na imagem adicionamos uma margem inferior (mp-4) e arredondamos as bordas através da classe do Bootstrap "rounded".

Incrementando o conteúdo

Ajustes no Breadcrumb

Como estamos em uma página interna, é importante que o Breadcrumb informe corretamente a localização da página e indique um link para facilitar a navegação.

Seção de comentários

Para que os usuários possam comentar a postagem, é necessário adicionar um pequeno formulário com um campo de texto e um botão. Para isso, criaremos um Card que conterá o formulário com os campos necessários.

Abaixo do formulário de comentários, vamos adicionar uma listagem com os últimos comentários realizados na postagem. Para isso, utilizaremos um novo componente do Bootstrap chamado de Media List.

Incrementando um pouco mais

Adicionando um vídeo

Para adicionar um video, podemos utilizar a tag <video> do HTML5. No entanto, é importante que esse vídeo se adapte à resolução do dispositivo do usuário. Para isso, é possível utilizar algumas classes utilitárias do Bootstrap.

Adicionando uma tabela

Por fim, vamos adicionar uma tabela no conteúdo de nossa postagem. Para adicionar uma tabela, basta utilizar a tag do HTML <table>, conforme já vimos em aulas anteriores. Além disso, vamos adicionar Badges à tabela.
