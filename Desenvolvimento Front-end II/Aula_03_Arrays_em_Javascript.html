Aula 03 - Arrays em Javascript

Postado por Monica Lima de Paiva

postado em 26 de Outubro de 2022

Criação de arrays em Javascript

Criamos um array envolvendo seus itens com colchetes e separando seus itens por vírgula:

        var arrayHomogenea = ['segunda', 'terça', 'quarta'];
        var arrayHeterogeneo = ['domingo', 3, true];

Acessando elementos de um array

Cada elemento em um array tem uma posição numérica chamada índice. Nós podemos acessar itens individualmente em um array usando o seu índice.

        const cidades = ['Natal', 'Fortaleza', 'Recife'];
        var minhaCidade = cidades[0];

        console.log(minhaCidade)
        console.log(cidades[2]);

Atualizando itens de um array

Uma vez que você tem acesso ao elemento em um array, pode atualizar o valor armazenado.

        let cursosTecnicos = ['web', 'eletrônica', 'redes'];

        cursosTecnicos[1] = 'jogos';
        console.log(cursosTecnicos); // Saída: ['web', 'jogos', 'redes']

Propriedade length

Por fim, uma das propriedades nativas do array é a propriedade length, que retorna o seu comprimento, ou seja, o número de elementos armazenados no array em questão.

        const listaDeCompras = ['ovos', 'leite', 'manteiga'];
        console.log(listaDeCompras.length);

        const novaListaDeCompras = ['ovos', 'leite', 'manteiga', 'chocolate', 'pão'];
        console.log(novaListaDeCompras.length);

Métodos push() e pop()

O método push() permite que você adicione itens no fim de um array.

Dica: Note que o método push() de fato modifica diretamente o array na qual o método é chamado. Por esse motivo, o método push() é considerado um método destrutivo, pois modifica o array original.

        const objetivos = ['estudar javascript', 'passar na disciplina'];
        objetivos.push('trabalhar com TI', 'relaxar');
        console.log(objetivos); // [ 'estudar javascript', 'passar na disciplina', 'trabalhar com TI', 'relaxar' ]

Já o método pop() remove o último item de um array. Note que o método pop() retorna o elemento removido.

        const objetivos = ['estudar javascript', 'passar na disciplina', 'trabalhar com TI'];
        var itemRemovido = objetivos.pop();
        console.log(objetivos); // [ 'estudar javascript', 'passar na disciplina' ]
        console.log(itemRemovido); // 'trabalhar com TI'

Métodos shift() e unshift()

O método shift() remove o primeiro elemento de um array. Funciona de maneira similar ao pop(), porém ao invés de remover o último item da lista, ele remove o primeiro item.

Já o método unshift() permite adicionar itens ao início de um array. Funciona de maneira similar ao push(), porém ao invés de adicionar um item no fim da lista, ele adiciona ao início dela.

Métodos splice() e slice()

O método splice() nos permite adicionar ou remover elementos no meio, ou em qualquer posição do array. Esse método modifica o array original. 

            const mensagens = ['Mensagem 1', 'Mensagem 2', 'Mensagem 3'];

            mensagens.splice(1, 1); // (inicio da manipulação, quantos elementos após o indice inicial)
            mensagens.splice(1, 0, 'Nova Mensagem'); // (inicio da manipulação, não remover, inserir elemento no seguinte indice)
            mensagens.splice(2, 1, 'Nova Mensagem 3'); // (inicio da manipulação, remover o numero de elementos (1), inserir o elemento)

O método slice() retorna um array contendo a cópia de uma parte de um array original, selecionado de um índice inicial até um final (final não incluso). Uma observação importante é que o array original não é modificado. 

        const letras = ['a', 'b', 'c', 'd', 'e', 'f'];
        let resultado = letras.slice(1, 4);
        console.log(resultado); // [ 'b', 'c', 'd' ]

Concatenando arrays

O método concat() permite concatenar dois ou mais arrays. Note que o retorno da função corresponde às arrays concatenadas. 

            const array1 = ['a', 'b'];
            const array2 = ['c', 'd'];
            const array3 = ['e', 'f'];

            const resultado = array1.concat(array2).concat(array3);

            console.log(resultado); // [ 'a', 'b', 'c', 'd', 'e', 'f' ]

Iterando arrays com forEach()

Esse é um método em que se executa uma função a cada iteração de um array. Portanto o método forEach() recebe como argumento uma função, também chamada de função de callback, com dois parâmetros: o valor e o índice do elemento.

        const array = ['João', 'Maria', 'Daniel', 'Alice'];

        array.forEach((elemento, indice) => { array[indice] = elemento.toUpperCase(); });

        console.log(array);  // [ 'JOÃO', 'MARIA', 'DANIEL', 'ALICE' ]

Métodos every() e some()

O método every() retorna true ou false, baseado na condição estabelecida e em todos os elementos do array em questão. É uma maneira interessante de rapidamente checar se todos os elementos dentro de um array satisfazem um determinado critério.

O método some() também retorna true ou false, porém para retornar true é suficiente que ao menos um dos elementos do array satisfaça o critério estabelecido pela função passada como argumento.

        let numeros = [12, 19, 7, 3, 9, 7, 3, 11];
        let todosMaioresQueTrinta = numeros.every(n => {return n < 30});
        console.log('Todos os números são maiores que 30? ' + todosMaioresQueTrinta); // false

        let algumMaiorQueTrinta = numeros.some(n => {return n > 30});
        console.log('Tem algum número maior que 30? ' + algumMaiorQueTrinta); // true

Métodos filter(), map() e reduce()

O filter() é um método que permite a criação de um novo array baseado nos elementos de um array preexistente que satisfizerem o critério estabelecido na função passada como argumento da função filter(). Em outras palavras, o método filter() cria um novo array com todos os elementos que passaram no teste implementado pela função fornecida.

O método map() cria uma nova array composta dos resultados da execução da função passada como argumento em cada um dos elementos do array original.

        const palavras = ['universidade', 'web', 'javascript', 'disciplina', 'aluno', 'nota'];
        const resultado1 = palavras.filter(palavra => palavra.length > 6);
        console.log(resultado1); // [ 'universidade', 'javascript', 'disciplina' ]

        const array = [1, 4, 9, 16];
        const resultado2 = array.map(x => x * 2);
        console.log(resultado2); // [ 2, 8, 18, 32 ]

o método reduce() executa uma função reducer (provida por você) para cada membro do array, resultando num único valor de retorno. Para simplificar, uma função reducer é uma função agregadora. Pode ser para fazer uma soma, encontrar o maior ou menor valor numa lista, entre muitas outras possibilidades. O reducer sempre percorrerá toda a lista e retornará apenas um resultado.

        const precos = [7.50, 32.00, 12.50, 27.50, 16.50, 9.90];

        const maiorPrecoReajustado = precos
          .filter((produto) => { return produto > 15.00 })
          .map((produto) => { return produto - 0.1*produto })
          .reduce((a, b) => { return Math.max(a, b) } );                            

        console.log(maiorPrecoReajustado); // 28.8

Ordenando arrays

Se quisermos ordenar de maneira descendente, basta invocar o sort() normalmente para ordenar de maneira crescente e chamar o método reverse() para reverter a ordem do array em questão.

Buscando em arrays com indexOf(), find() e findIndex()

Método indexOf() permite encontrar o índice da primeira ocorrência do valor passado como parâmetro.

O método find() retorna o valor da primeira ocorrência no array que satisfaça a função passada como argumento.

Note que o find() é bastante similar ao filter(), porém o find() para a execução e retorna somente o primeiro valor que satisfaça a condição estabelecida pela função.

O método findIndex() retorna o índice (posição) da primeira ocorrência no array que satisfaça a função passada como argumento.

Outra maneira é através dos métodos find() e findIndex(). Funcionam de maneira similar, porém o primeiro retorna o valor e o segundo retorna o índice da primeira ocorrência no array que satisfaça a função passada como argumento. Note que o find() é bastante similar ao filter(), porém o find() para a execução e retorna somente o primeiro valor que satisfaça a condição estabelecida pela função.

        const mensagens1 = ['Mensagem 1', 'Mensagem 2', 'Mensagem 3', 'Mensagem 2'];
        var busca = mensagens1.indexOf('Mensagem 2');
        console.log(busca); // 1

        const array1 = [5, 12, 8, 130, 44];
        const valor = array1.find(item => item > 10);
        console.log(valor); // 12

        const array2 = [5, 12, 8, 130, 44];
        const indice = array2.findIndex(item => item > 10);
        console.log(indice); // 1

        const mensagens2 = ['Mensagem 1', 'Mensagem 2', 'Mensagem 3', 'Mensagem 2'];
        var busca = mensagens2.indexOf('Mensagem 2');
        console.log(busca); // 1
