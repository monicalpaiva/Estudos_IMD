Aula 05 - Manipulação do DOM

Postado por Monica Lima de Paiva

postado em 26 de Outubro de 2022

O que é DOM

DOM é a abreviação para Document Object Model. É uma representação dos elementos HTML de uma página de uma forma organizada e hierárquica, no formato de uma árvore. Esse é um dos conceitos mais importantes quando precisamos interagir com a página web a partir do Javascript. Por exemplo, para atualizar elementos, criar e até mudar texto do HTML pelo JavaScript nós utilizamos a DOM. Por isso é comum escutarmos que precisamos manipular o DOM com o Javascript para obter os nossos objetivos. Porém apesar da forte ligação com a linguagem Javascript, ele não é uma parte dela, e podemos pensar no DOM como uma interface que pode ser acessada pelo JavaScript.

Acessando elementos do DOM

document.getElementById()

Talvez a maneira mais simples de acessar um único elemento do DOM é pelo seu ID, um atributo que pode ser utilizado em elementos HTML. Obviamente, para acessar um elemento dessa maneira, é necessário que esse nó tenha um atributo ID.

document.getElementsByClassName()

Retorna todos os elementos com a classe especificada como argumento.

document.getElementsByTagName()

Retorna todos os elementos no documento com a tag especificada como argumento.

document.querySelector()

Retorna o primeiro elemento no documento que satisfaça o seletor indicado como argumento.

document.querySelector('#teste-query');

Retorna todos os elementos no documento que satisfaçam o seletor indicado como argumento.

document.querySelectorAll('.teste-query-all');

IMPORTANTE:Só para revisar, quando queremos nos referir a uma ID, usamos o prefixo jogo da velha (#). Quando queremos nos referir a uma classe, usamos o prefixo ponto-final (.). Caso queiramos nos referir a uma tag, basta não incluir nenhum prefixo. E existem diversas outras variações que podemos fazer usando seletores CSS, trata-se de uma ferramenta bastante poderosa.

Navegando através do DOM

Nós podemos acessar o elemento pai de um nó a partir da propriedade parentNode.

document.querySelector('p').parentNode;

A propriedade childNodes retorna uma lista com todos os nós filhos do nó em questão:

document.querySelector('body').childNodes;

Caso você queira retornar somente com elementos, podemos usar a propriedade children.

document.querySelector('body').children;

Por fim, temos os nós irmãos, que são os nós que estão no mesmo nível de hierarquia na árvore DOM e abaixo do mesmo nó pai. Para navegar entre os nós irmãos, independentemente do seu tipo, seja elemento ou texto, por exemplo, utilizamos as propriedades nextSibling e previousSibling, que representam, respectivamente, o próximo e o anterior irmão do nó em questão. Caso queiramos filtrar somente elementos HTML, usamos as propriedades nextElementSibling e previousElementSibling.

const teste = document.querySelector('#item').children[1]; // Acessando o segundo filho do elemento de id 'item'

teste.nextElementSibling.style.background = 'orange'; // Mudando a cor de background do próximo irmão do elemento de id 'item'

Modificando elementos do DOM

Criando novos elementos HTML com createElement

const paragrafo = document.createElement('p');

Adicionando texto a um elemento HTML com textContent

paragrafo.textContent = "Eu sou um parágrafo!";

Adicionando código HTML a um elemento com innerHTML

paragrafo.innerHTML = "Eu sou um parágrafo em negrito!";

Inserindo nós no DOM com appendChild

            const lista = document.querySelector('ul');
            const novoItem = document.createElement('li');
            novoItem.textContent = 'Hello World!';

            lista.appendChild(novoItem); // Adicionando o novo item ao fim da lista

Inserindo nós no DOM com insertBefore

document.body.insertBefore(document.createElement("p"), document.querySelector("a"));

Removendo nós do DOM com removeChild e remove

          list.removeChild(list.lastElementChild); // remove o elemento filho indicado como argumento
          list.children[2].remove(); // remove o próprio elemento selecionado

Acessando, removendo e modificando atributos

          const image = document.querySelector('img');

          image.hasAttribute('src'); // true
          image.getAttribute('src'); // retorna o link
          image.removeAttribute('src'); // remove o atributo src
          image.setAttribute('src', 'https://www.imd.ufrn.br/logo.png'); // Atribui a imagem da URL ao atributo 'src' do elemento

Modificando estilos CSS de elementos do DOM com style

          const div = document.querySelector('div');

          div.style.borderRadius = '50%';
          div.style.backgroundColor = 'lightgreen';
          div.style.fontSize = '30px';

Modificando classes de elementos do DOM com classList

            const activeDiv = document.querySelector('.active');

            activeDiv.classList.add('hidden'); // adiciona a classe 'hidden' ao elemento
            activeDiv.classList.remove('hidden'); // remoe a classe 'hidden' do elemento
            activeDiv.classList.toggle('hidden'); // alterna entre a adição e a remoção da  classe 'hidden'
            activeDiv.classList.replace('active', 'alert'); // Substitui a classe 'ative' pela classe 'alert'
