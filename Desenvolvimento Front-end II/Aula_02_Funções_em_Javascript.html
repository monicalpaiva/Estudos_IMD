Aula 02 - Funções em Javascript

Postado por Monica Lima de Paiva

postado em 26 de Outubro de 2022

Declarando funções

    Essa declaração de função consiste em:
  • A palavra-chave function;
  • O nome da função, ou seu identificador, seguido de parênteses;
  • O corpo da função, ou o bloco de instruções necessárias para desempenhar uma tarefa, delimitado pelas chaves.

Chamando uma função

Para chamar uma função no seu código, é necessário incluir o nome da função seguida de parênteses. Uma característica interessante do Javascript é a possibilidade de realizar chamadas de função antes da efetiva declaração dessa mesma função.

            function olaMundo() {
                console.log('Olá Mundo!');
             }
             
             olaMundo(); // chamando a função aqui!

Parâmetros e argumentos

Parâmetros permitem que funções aceitem entradas e manipulem essas entradas para desempenhar alguma tarefa. Usamos parâmetros para definir qual a informação deve ser passada para a função quando ela é chamada.

Os valores que são passados para uma função quando ela é chamada são chamados argumentos. Argumentos podem ser passados como valores ou como variáveis.

Parâmetros padrão

Ao utilizar parâmetros padrão, nós lidamos com situações em que um argumento não é passado para uma função que está esperando por um argumento.

            function ola(nome) {
                console.log('Olá, ' + nome + '!');
            }
            
            ola('Maria'); // retorna "Olá Maria!"
            ola(); // retorna "Olá undefinied!"

Para correção desse "undefinied" podemos editar como valor padrão:

            function ola(nome = 'Estranho') {
                console.log('Olá, ' + nome + '!');
            } 

        ola(); // retorna "Olá Estranho!"

Retornos

Quando uma função é chamada, o computador irá executar o corpo dessa função e avaliar o resultado daquela chamada de função. Se não for explicitamente realizada a partir de uma instrução return, por padrão, o resultado é undefinied. Sempre em uma função, mesmo que tenha várias chamadas de retorno, quando a mesma encontrar o primeiro return ela será encerrada.

Funções auxiliares

Nós podemos usar o valor de retorno de uma função dentro de outra função. Essas funções que são chamadas dentro de outras funções são comumentes chamadas de funções auxiliares.

Nós podemos usar funções para segmentar pequenas tarefas quando precisamos. Escrever funções auxiliares pode ajudar a quebrar tarefas grandes e complexas em tarefas menores e mais simples!

Expressão de função e funções flechas

Outra forma de definir uma função é com a chamada expressão de função. Ela pode ser útil quando precisamos definir uma função dentro de alguma expressão. Por exemplo, para armazenarmos uma expressão de função dentro de uma variável

        var eFimDeSemana = function(dia) {
            if (dia === 'Sábado' || dia === "Domingo") {
                return true;
            } else {
                return false;
            }
        }
        
        console.log(eFimDeSemana('Segunda-Feira'))

O Javascript também permite escrever funções usando a sintaxe conhecida como função flecha, ou arrow function. Esse tipo de sintaxe permite omitir a palavra-chave function, deixando a definição de uma função bem mais reduzida. 

            var calcularArea = (comprimento, altura) => {
                return comprimento * altura;
            }
            
            console.log(calcularArea(10, 6));
        
var olaComFuncoesFlechaCondensada = nome => console.log('Olá ' + nome + "!");
